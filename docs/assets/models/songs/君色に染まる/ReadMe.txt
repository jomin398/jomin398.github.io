君色に染まる(足太ぺんたさん)  MikuMikuDanceモーション

使用するにあたってオンライン規約をご確認ください。
http://ch.nicovideo.jp/2ntsun/blomaga/ar867730
特にネット上にアップする場合は必ず規約を遵守してください。

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

【内容物】
解凍フォルダ
├─モーション_小.vmd   ・・・モーフ変化量が小さいモデル用
├─モーション_大.vmd   ・・・モーフ変化量が大きいモデル用（配布動画使用）
├─カメラ.vmd          ・・・配布動画と同じカメラ
├─リップのみ.vmd      ・・・歌唱部分のみの口パクモーション
├─オンライン利用規約  ・・・利用規約へのショートカット
└─ReadMe.txt          ・・・このファイル


【使用ボーン、モーフ】
ボーン：標準、準標準ボーン
モーフ：[  目  ]まばたき、笑い、びっくり
        [リップ]あ、い、う、え、お、にやり
        [ まゆ ]にこり、怒り、困る、上、下
目線  ：両目

※グルーブボーンで上下移動していますのでモデルに実装されていないとただの水平移動になります。
  また腕・肘の捩りボーンを使用しています。これも実装されていないと怪しい動きになります。
  PMDEのプラグインで簡単に追加できるのでVPVPwikiなど参考にしてください。
  モデルを改造する場合は規約に反しないようご注意ください。


【使用上の注意】
●モーションは原曲に合わせています。
  ただし音源のDL方法によってはズレる場合があるようなので、
  動画投稿する場合一旦出力・エンコードして確認してみるなど対策を行ってください。
  ※口パクは配布動画で使用している音源（じろ様）にあわせた動きになっています。
  
●モーションは表情モーフ変化量の大小の違いだけで、ダンスモーションは同じです。
