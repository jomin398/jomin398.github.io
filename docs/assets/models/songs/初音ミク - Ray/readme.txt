＜モーションデータについて＞

★このモーションは2mkさんのrayで使用されたモーションデータを更に調整したものです。

★必要ボーン
標準ボーン+
　・グルーブ
　・上半身2
　・肩P
　・腕捩
　・手捩
　・親指0

★表情データについて
配布されるデータは2mkさんの動画で使われている表情データではありません。「両目」の動き含め私が配布用に新たに作成したものになります。

おまけフォルダに入っている2mkさんVerのモーションにこの表情データを入れてもいい感じにはなりますが、
両目がカメラによっては目立つかもしれないのでそちらを使用する場合は要注意してください。

使用したモーフ(YYB式)
口：「あ」「い」「う」「え」「お」「にやり」
目：「まばたき」「笑い」「ウィンク右」「ｷﾘｯ」
眉：「にこり」「真面目」「困る」「上」「下」

★カメラについて
改変自由というか改変推奨します　特に観客側を映してるところ

★おまけデータについて
「ray_2mkさんVer_配布用調整版」
2mkさんが使用されたモーションデータほぼそのまんまで微調整し、カメラに映っていなかった足を整えたモーションデータです。

「ray_足IK化」
SOLAVerのrayモーションをmiuさんにお借りしたツールで足をIK化して少し調整したものです。足の動きがちょっと微妙になっちゃうので、
どうしてもセンターを動かしたい時足やIKにしたい時に使ってください
ちょっとプルプルしてるところもありましたが、見なかったことにしました←

●踊り手さんと体型の似ているYYB式ミクを採用しているだけで「YYB式用モーション」としてキャプチャーしたわけではないですが、
2mkさんに合わせて「YYB式ミク10th」でモーション修正を行っているので、
Vmdサイジング等でモーション変換する場合は変換元モデルを「YYB式10th」にすれば大丈夫だと思います。
